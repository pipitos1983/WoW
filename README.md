# WoW
WoW configs and addons

ResourceTracker - Аддон для отслеживания классовых ресурсов.(более не актуально)  
Простенькая замена стандартному с добавлением нужного функционала.  
Перемещение с shift + лкм.  
Сброс позиции на центр экрана: `/rst reset`  
Отключение/Включение полосы основного ресурса (мана, энергия и прочее): `/rst bar`
